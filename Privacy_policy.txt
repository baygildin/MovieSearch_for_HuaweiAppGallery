Privacy Policy

General Information

MovieSearch! is an application designed to help users find information about movies and TV shows using the OMDb API. This Privacy Policy explains how the app handles user data.

Data Collection

MovieSearch! does not collect any personal data from users. All information displayed in the app is sourced from the OMDb API.

Data Usage

Since the app does not collect any personal data, no user data is used for other purposes.

Data Sharing and Disclosure

MovieSearch! does not share user data with third parties. OMDb API is used solely as an information provider for movies and TV shows. The data retrieved from OMDb API is displayed to users and is not shared with any third parties. Google Firebase and other data storage services are not used in this app.

Data Security

MovieSearch! does not process or store sensitive user data. Since no personal data is collected or stored, no specific data security measures are required.

User Rights

Users retain their usual privacy rights. Since MovieSearch! does not store any user data, rights regarding access, modification, or deletion of data do not apply.

Policy Changes

We may update this Privacy Policy from time to time. The updated policy will be available in the app or on our website for users to review.
